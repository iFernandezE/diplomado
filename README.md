# PROYECTO DIPLOMADO

### 1. Clonar el proyecto
```bash
git clone https://github.com/iFernandezE/diplomado.git
```

### 2. Instalar las dependencias
```bash
npm install
```

### 3. Copiar variables de entorno y configurar
```bash
cp .env.example .env
```

### 4. Levantar el proyecto
### 4.1 En desarrollo
```bash
npm run dev
```

#### 4.2 En produccion
```bash
npm run start
```
